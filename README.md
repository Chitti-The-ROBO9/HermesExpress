# HermesExpress 
### The Hyper-Local Mobility & Travel Orchestrator

**HermesExpress** is a decision-engine dashboard built for real-world travel constraints. Unlike standard navigation apps, it bridges the gap between fuel levels, budgets, and intermodal transport.

## Key Features
- **Smart Radius Engine:** Visualizes fuel-range perimeters based on vehicle mileage.
- **Intermodal Cost Comparison:** Real-time ₹ comparison between personal vehicle use and ride-hailing (Rapido/Ola).
- **Deep-Link Execution:** One-tap ride booking using OS-level URI intents.
- **Scale-Ready UI:** Integrated Flight and Hotel discovery modules for Phase 3 expansion.

##  Tech Stack
- **Frontend:** Leaflet.js, Tailwind CSS, Mapbox API
- **Backend:** Python (Flask), Foursquare Places API
- **DevOps:** Environment-variable security & CI/CD ready structure.
